Differentiation

Tags Similarity Difference
hr & br create break (to the next line)   hr : Has break & line (use when there is a change of content)
  br : Only break, no line used only to go to next line
ul & ol create list   ul : Bullets (or other style) but not numbered/ordered
  ol : Numbered (ordered list) used when list needs some kind of order
select & checkbox & radiobutton for selecting options   select : Dropdown list to select from
  checkbox : Square box that can be ticked to select
  radio button : circular can be highlighted
div & span wrap content   div : Block-line (line break before & after)
  span : Text in the same line (in-line text)
script & style use to provide places to write languages other than HTML   style : for writing CSS
  script : for writing Javascript
