CSS syntax consists of selector, property, and value
  1. Inline
  2. Internal
  3. External
 CSS selectors are patterns used to select the HTML element(s) you want to style. There are four different CSS combinators.
  1. descendant selector (space)
  2. child selector (>)
  3. adjacent sibling selector (+)
  4. general sibling selector (~)

Example of CSS Syntax

body {
        font-size: 16px;
    }

Example of CSS Types

  1. <div style="color: red;font-size = 2rem;">Inline</div>
  2. <style> div { color:blue; font-size: 3rem; } </style>
  3. div { color:blue; font-size: 3rem; }

  3. div + a
