The difference between property and value.

Property -> The property that determines the style of a selected HTML element

Value -> The given values to the CSS property
The categories of CSS which were taught in the pre-camp.
  1. property for content: color , size , font , text, list-style-type, Height, Width, background
  2. property for border : border, table
  3. property for padding : padding (top right buttom left)
  4. property for margin : margin (top right buttom left)
  5. property for element : display, Height, Width, Position, Float, Align, Flex, Grid
CSS values and units
  • relative length units

    • Relative length units specify a length in relation to something else such as em and rem.

  • absolute length units

    • Absolute length units are fixed to a physical length such as px, cm and mm.
